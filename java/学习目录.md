### 学习案例（带讲解的独立案例）

- [**类与封装 - Book 示例**](学习案例/类与封装-Book.md)：类、对象、构造方法、封装（private/getter/setter）、@Override 说明，写一个简单的 Book 类并运行 BookDemo。
- [**继承与重写 - Animal -> Dog 示例**](学习案例/继承与重写-Animal-Dog.md)：继承、super、重写，写父子类 Animal -> Dog 并运行 InheritanceDemo。
- [**接口与多态 - Shape 示例**](学习案例/接口与多态-Shape.md)：接口、两个实现类（Circle、Rectangle）、多态调用，运行 InterfaceDemo。
- [**数组与 List 的几种定义**](学习案例/数组与List的几种定义.md)：int[]、Arrays.asList、new ArrayList 的区别与使用场景。
- **练习 - 计算器**：`basics/src/11-practice-calculator/`，含接口 BasicCalculator、实现 SimpleCalculator、子类 ScientificCalculator，设计思路见该目录下 `笔记-计算器设计思路.md`。
- [**集合 - ArrayList、HashMap**](学习案例/集合-ArrayList-HashMap.md)：增删改查、用 List 存多条数据，见 `12-collections/`。
- [**异常 - try-catch、throws**](学习案例/异常-try-catch-throws.md)：捕获、throws 声明、throw 抛出，见 `13-exception/`。
