# Java 基础（basics）

入门示例与常用 API 演示，按**文章/学习案例**分文件夹存放，便于对照阅读与运行。

> 总览与学习案例索引见上一级 [学习目录.md](../学习目录.md)。

---

## 目录与文章对应关系

| 文件夹 | 对应文章/主题 | 文件 |
|--------|----------------|------|
| **01-entry** | 入口与命令行参数 | `Main.java`, `Run.java` |
| **02-variables-control** | 变量与数据类型、控制流 | `BasicsDemo.java` |
| **03-array** | 数组、Arrays、Stream、工具类 | `ArrayDemo.java`, `ArrayUtils.java` |
| **04-method** | 方法（构造/实例/静态）、修饰符 | `MethodDemo.java` |
| **05-string** | 字符串常用 API、StringBuilder；练习：拼接/截取/判断 | `StringDemo.java`, `StringPractice.java` |
| **06-number** | 数值、Math、包装类 | `NumberDemo.java` |
| **07-object** | 对象、equals、toString | `ObjectDemo.java` |
| **08-class-encapsulation** | 类、对象、构造方法、封装（Book） | `Book.java`, `BookDemo.java` |
| **09-inheritance** | 继承、super、重写（Animal -> Dog） | `Animal.java`, `Dog.java`, `InheritanceDemo.java` |
| **10-interface-polymorphism** | 接口、多态（Shape + Circle、Rectangle） | `Shape.java`, `Circle.java`, `Rectangle.java`, `InterfaceDemo.java` |
| **11-practice-calculator** | 练习：类+继承+接口（计算器） | `BasicCalculator.java`, `SimpleCalculator.java`, `ScientificCalculator.java`, `CalculatorDemo.java`，笔记见 `笔记-计算器设计思路.md` |
| **12-collections** | 集合：ArrayList、HashMap 增删改查；List 存多条数据 | `ArrayListDemo.java`, `HashMapDemo.java` |
| **13-exception** | 异常：try-catch、throws、throw | `ExceptionDemo.java` |

## 运行方式

在 `full-stack/java/basics` 目录下执行。每个文件夹内的类可单独编译运行，依赖同目录下其他类时一起编译。

```bash
cd full-stack/java/basics

# 入口
javac src/01-entry/Main.java -d out && java -cp out Main

# 数组（ArrayDemo 依赖 ArrayUtils）
javac src/03-array/*.java -d out && java -cp out ArrayDemo

# 方法
javac src/04-method/MethodDemo.java -d out && java -cp out MethodDemo

# 类与封装（Book）
javac src/08-class-encapsulation/*.java -d out && java -cp out BookDemo

# 继承（Animal -> Dog）
javac src/09-inheritance/*.java -d out && java -cp out InheritanceDemo

# 接口与多态（Shape）
javac src/10-interface-polymorphism/*.java -d out && java -cp out InterfaceDemo

# 练习：计算器
javac src/11-practice-calculator/*.java -d out && java -cp out CalculatorDemo

# 集合
javac src/12-collections/*.java -d out && java -cp out ArrayListDemo
javac src/12-collections/*.java -d out && java -cp out HashMapDemo

# 异常
javac src/13-exception/*.java -d out && java -cp out ExceptionDemo
```

其他单文件 demo（如 StringDemo、NumberDemo、ObjectDemo、BasicsDemo）同理：`javac src/xx-xxx/*.java -d out && java -cp out 类名`。

在 Cursor 中打开 **java/basics** 文件夹即可运行、调试。
