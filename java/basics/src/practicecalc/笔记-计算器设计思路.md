# 计算器练习：如何考虑、设计思路（后端小白向）

代码在 **`src/practicecalc/`**，包名 **`practicecalc`**（不能用 `11-practice-calculator` 这类以数字开头的目录当包名，否则 Cursor/VS Code 的 Java 插件会报大量「包声明与目录不匹配」等错误）。

## 运行

```bash
cd full-stack/java/basics
javac -d out src/practicecalc/*.java
java -cp out practicecalc.CalculatorDemo
```
