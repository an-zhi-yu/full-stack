# full-stack 全栈学习仓库

本仓库用于全栈学习与练手项目，包含前端、后端、Java/Python/Linux 学习与示例。

## 文档索引

- **仓库目录说明.md**：目录划分、各文件夹用途。
- **项目结构与技术栈.md**：整体结构、打包上线、前端技术栈方案。
- **java/环境与搭建.md**：Mac 上 Java 开发环境与 Cursor 使用。
- **java/编码规范.md**：Java 项目规范（本仓库约定）。
- **java/常用API速查.md**：Java 常用 API 速查。
- **frontend/blog/前端代码规范.md**：博客前端代码规范（React + Ant Design 等）。

## 快速开始

- **前端博客**：`cd frontend/blog && pnpm install && pnpm dev`
- **Java 示例**：见 `java/basics/说明.md`，需先安装 JDK 17（见 `java/环境与搭建.md`）。

目的：前端开发 → AI 全栈 / Java 后端
要求：代码用常见行业写法，注释详细、从前端角度类比
