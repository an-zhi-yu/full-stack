# 前端项目目录

前端代码写在本目录下的**各个子项目**里，每个子项目一个文件夹，独立 `package.json`、独立构建。

## 当前/计划项目

| 目录     | 说明           |
|----------|----------------|
| `blog/`  | 博客前台（React + Ant Design + TypeScript） |
| `admin/` | 示例：后台管理（可新建） |

## 新建一个前端项目

```bash
cd full-stack/frontend
pnpm create vite 项目名 --template react-ts
cd 项目名 && pnpm install
```
