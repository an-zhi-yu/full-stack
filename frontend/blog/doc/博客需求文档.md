# 博客需求文档

> 本文档作为博客开发的需求总库，每个迭代从此处取需求逐条设计实现。
> 优先级：🔴 P0 核心必做 / 🟠 P1 重要 / 🟡 P2 加分 / ⚪ P3 长远规划

## 一、基础架构

### 1.1 前端技术栈
| 类别 | 选型 |
|------|------|
| 框架 | React 18 |
| 路由 | React Router 6 |
| 状态 | Zustand |
| UI | Ant Design 5 |

| 构建 | Vite |

## 二、首页（Home）

### 2.1 Hero 区 🔴
- 浮动光球背景动画（CSS Keyframes）
- 渐变色大标题
- 副标题描述文案
- 两个 CTA 按钮：「浏览文章」「关于我」
- 进入动画（fadeUp + 延迟）

### 2.2 统计栏 🟠
- 展示：文章总数、分类数、标签数、学习天数
- 玻璃形态背景
- 渐变色数字

### 2.3 精选文章 🔴
- 置顶/精选文章卡片网格（2-3 列）
- 每卡片展示：分类、标题、副标题、日期、阅读时间、标签
- 悬停：上浮 + 边框发光效果

### 2.4 技术分类 🔴
- 分类卡片网格（icon + 名称 + 描述）
- 悬停：对应主题色渐变背景
- 点击跳转到对应分类页

### 2.5 最新文章 🔴
- 列表式展示最新 6 篇
- 每项：序号（渐变色）、标题、副标题、分类+日期
- 悬停：整行右移 + 箭头变色

### 2.6 其他
- 🟡 「推荐文章」轮播/横向滚动
- 🟡 订阅邮件 / RSS 入口
- ⚪ 访客统计（PV/UV）展示
- ⚪ GitHub 贡献图展示

## 三、文章列表页（Posts）

### 3.1 基础功能 🔴
- 所有文章分页展示（每页 10 条）
- 每卡片展示：标题、副标题、分类、标签、日期、阅读时间
- 按发布日期降序排列

### 3.2 筛选与搜索 🟠
- 按分类筛选（标签式按钮，支持多选）
- 按标签筛选
- 关键词搜索（前端内存搜索）
- URL 参数记录筛选状态（刷新保持）

### 3.3 排序 🟡
- 最新 / 最旧 / 最热（阅读量）

### 3.4 样式
- 卡片式（默认）/ 列表式 切换按钮 🟡
- 无结果提示

## 四、文章详情页（Post Detail）

### 4.1 文章内容渲染 🔴
- 标题、副标题、分类、标签、日期、阅读时间
- 内容块渲染（heading/paragraph/code/list/tip/warning/table）
- 代码块：语言标签 + 说明 + 代码（等宽字体，深色背景）
- 行内代码高亮
- 表格：条纹行、悬停高亮
- 提示框 (💡 tip) / 警告框 (⚠️ warning)

### 4.2 代码高亮 🟠
- 引入 highlight.js 或 shiki 实现语法着色
- 支持 Java、TypeScript、JavaScript、Python、Bash、SQL

### 4.3 目录（TOC） 🔴

- 点击跳转到对应锚点（smooth scroll）
- 🟡 滚动时高亮当前所在章节

### 4.4 相关文章 🔴
- 同分类其他文章（侧边栏）

### 4.5 导航 🔴
- 上一篇 / 下一篇 导航按钮
- 面包屑：首页 > 分类 > 文章

### 4.6 阅读体验 🟠
- 阅读进度条（顶部横向进度条）
- 「回到顶部」按钮
- 估算阅读时间（已有）
- 复制代码按钮（代码块右上角）

### 4.7 互动 🟡
- 文章点赞（localStorage 持久化）
- 评论系统（接入 Giscus 或 Utterances）

## 十二、开发规范

### 12.1 目录约定
- 页面：`src/pages/<页面名>/index.tsx`（每页一目录）
- 组件：`src/components/<类别>/<组件名>.tsx`
- 样式：同名 `index.module.less`（每组件/页面）
- 数据：`src/data/<分类>/`（按分类分文件）
- Hooks：`src/hooks/`
- 工具：`src/utils/`

### 12.2 样式
- 样式统一用 **LESS + CSS Modules**
- 全局 CSS 变量在 `src/styles/global.less`（支持主题切换）
- LESS 变量 + Mixins 在 `src/styles/variables.less`（自动全局注入）
- 响应式断点在 variables.less 中定义统一 mixin

### 12.3 提交流程
- Husky + lint-staged：提交前自动 lint + format
- 提交信息格式：`feat/fix/style/refactor/docs: 描述`
