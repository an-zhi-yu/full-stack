# 博客前端

React + React Router + Zustand + Ant Design + Axios + TypeScript，带 ESLint、Prettier、Husky、Vitest。

## 运行

```bash
# 在 full-stack/frontend/blog 目录下
pnpm install   # 或 npm install
pnpm dev       # 或 npm run dev
```

## 脚本

| 命令 | 说明 |
|------|------|
| `pnpm dev` | 开发环境 |
| `pnpm build` | 打包 |
| `pnpm lint` | 检查代码规范 |
| `pnpm format` | 格式化代码 |
| `pnpm test` | 运行测试（监听） |
| `pnpm test:run` | 单次运行测试 |
| `pnpm test:coverage` | 测试覆盖率 |
